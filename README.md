# 🌍 LangBase – Language Translation System

A JavaFX desktop application that allows users to translate text between multiple languages with user authentication and history tracking.

---

## 🚀 Features
- User Signup & Login
- OTP-based Password Recovery
- Real-time Language Translation
- Translation History stored in MySQL
- User Profile Management
- Email notifications

---

## 🛠️ Tech Stack
- JavaFX
- Java (JDBC)
- MySQL Database
- CSS for UI styling

---

## 📸 Project Highlights
- Modern GUI using JavaFX
- Secure login system
- Multi-language support
- Clean UI design
